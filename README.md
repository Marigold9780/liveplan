# LivePlan

给朋友内部使用的日音 Live 规划助手原型。核心体验是像逛商城一样浏览官方已公布 Live，把感兴趣的场次加入自己的意向列表，再和朋友一起判断是否要冲。

## 打开方式

直接用浏览器打开：

`index.html`

当前版本不需要安装依赖，也不需要启动服务。数据会保存在浏览器 `localStorage` 里。

## 第一版功能

- Live 总览：跟进数量、已确定数量、临近截止、可加入发现数量
- 官方发现池：展示 BanG Dream! 已公布且尚未演出的 Live
- 意向 cart：从发现池把 Live 加入自己的规划列表
- Live 管理：新增、编辑、删除演出
- 筛选搜索：按艺人、城市、场馆、状态查找
- 朋友协同：添加成员，给每场 Live 标记“想去 / 观望 / 不去”
- 提醒视图：按抽选或开票截止日期排序
- 数据导出：导出当前数据为 JSON

3. 增加账号和云端数据库，多人可以实时共享。

## 数据来源说明

当前原型内置了 BanG Dream! 官方页面中已公布的部分未演出 Live，用于验证“官方发现池 -> 加入意向”的产品流程。

静态网页不能稳定地直接跨站抓取官网页面。真正上线时建议增加一个轻量后端或定时脚本，负责抓取官网、解析结构化数据、去重更新，然后把结果提供给前端。
