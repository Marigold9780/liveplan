# LivePlan

给朋友内部使用的日音 Live 规划助手原型。

## 打开方式

直接用浏览器打开：

`index.html`

当前版本不需要安装依赖，也不需要启动服务。数据会保存在浏览器 `localStorage` 里。

## 第一版功能

- Live 总览：跟进数量、已确定数量、临近截止、预算合计
- Live 管理：新增、编辑、删除演出
- 筛选搜索：按艺人、城市、场馆、状态查找
- 朋友协同：添加成员，给每场 Live 标记“想去 / 观望 / 不去”
- 提醒视图：按抽选或开票截止日期排序
- 数据导出：导出当前数据为 JSON
